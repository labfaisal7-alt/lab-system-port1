# بوابة المختبر

نسخة React أولية لواجهة بوابة المختبر داخل البوابة الداخلية للمستشفى.

## التشغيل

```powershell
& 'C:\Program Files\nodejs\npm.cmd' install
& 'C:\Program Files\nodejs\npm.cmd' run dev
```

ثم افتح الرابط المحلي الذي سيظهر لك في الطرفية.

## الوحدات الحالية

- إحصائيات أقسام المختبر
- جرد المحاليل والمواد
- سياسات ومستندات المختبر

## الهيكل

- `index.html` نقطة الدخول
- `src/main.jsx` تشغيل React
- `src/App.jsx` الواجهة الرئيسية
- `src/styles.css` التنسيق

## الخطوات القادمة المقترحة

- ربط النماذج بقاعدة بيانات أو API
- إضافة تسجيل دخول وصلاحيات
- دعم رفع الملفات للمستندات
- إضافة رسوم بيانية ولوحة مؤشرات
- إضافة تنبيهات لنقص المواد واقتراب انتهاء الصلاحية
